# MsgSniper
A simple overlay for discord that snipes messages and allows you to copy the username
<br></br>

# Usage
Copy the code and paste it into your discord browser console

# Features

- Whitelisted words
- Banned words
- Copy the username of the sender
- Minimized & delete buttons
